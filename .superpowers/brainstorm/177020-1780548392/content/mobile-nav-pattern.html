A — Bottom Tab Bar

Thanh tab cố định ở đáy màn hình. List → Chat → Info là back-stack dọc. Pattern của WhatsApp, Messenger, Zalo.

Pros

  • Quen thuộc với user mobile
  • Điều hướng 1 tay dễ
  • Slot sẵn cho thêm tab

Cons

  • Tốn 50px chiều cao
  • Cần thiết kế thêm tab bar

B — Slide Drawer

Danh sách ẩn trong drawer trượt ra từ trái (giống Gmail). Chat là màn hình chính. Bấm ☰ để mở drawer.

Pros

  • Tối đa diện tích chat
  • Không tốn tab bar

Cons

  • Ít trực quan hơn
  • Khó discover conv list
  • Cần gesture handler

C — Full Stack Navigation

3 màn hình độc lập: List → Chat → Info. Điều hướng bằng back button. Pattern của Telegram, Signal.

Pros

  • Đơn giản nhất, ít code nhất
  • Tái dụng component 100%
  • Tailwind breakpoint thuần túy

Cons

  • Không có persistent nav
  • Không có bottom tab (có thể thêm sau)

💡 Khuyến nghị của mình: C — Full Stack vì ít thay đổi nhất, tái dụng hoàn toàn component hiện có, và dễ bổ sung Bottom Tab Bar sau (option A) nếu cần.
